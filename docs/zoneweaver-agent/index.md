# Zoneweaver Agent

{: .no_toc }

The Zoneweaver Agent is the Bhyve/OmniOS host agent of the Hyperweaver platform: a Node.js service that manages zones, networking, storage, and system monitoring on a single OmniOS host, and serves the Hyperweaver UI directly for standalone (Direct-mode) installs.
